# sql-data-analysis-project
This project provides a collection of SQL scripts for data exploration, analytics and reporting. These scripts cover various analyses such as database exploration, measures & dimensions exploration, date_range_exploration, magnitude analysis, ranking analysis, change_over_time_analysis, performance analysis, data_segmentation_analysis, part_to_whole_analysis, consolidated product report and customer report. These scripts helps to prpeare good visulaization quickly.
